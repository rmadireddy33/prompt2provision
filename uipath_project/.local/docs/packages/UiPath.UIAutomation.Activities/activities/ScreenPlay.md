# ScreenPlay

`UiPath.Semantic.Activities.NUITask`

Performs a given UI task, using AI, on the attached application.

**Package:** `UiPath.UIAutomation.Activities`
**Category:** UI Automation.Application

## How to create a new ScreenPlay

To generate the default XAML for this activity, run the following command:

```bash
uip rpa get-default-activity-xaml --activity-class-name UiPath.Semantic.Activities.NUITask
```
## Notes

- This activity must be placed inside a **Use Application/Browser** (`NApplicationCard`) scope.
- The `Task` property accepts a natural language prompt describing what the AI should accomplish on the UI.
- Use `AgentType` to select the AI agent model for task planning and execution.
- `MaxIterations` controls the maximum number of steps the AI can take, preventing infinite loops.
- Enable `IsDOMEnabled` for web applications to improve element targeting accuracy using DOM data.
- The `IsVariableSecurityDisabled` option should only be enabled when prompt instructions must be passed via variable values.
