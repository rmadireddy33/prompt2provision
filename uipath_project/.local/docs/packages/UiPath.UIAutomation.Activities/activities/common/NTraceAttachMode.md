# NTraceAttachMode

`UiPath.UIAutomationNext.Enums.NTraceAttachMode`

When trace artifacts are attached to a job.

## Values

| Value | Description |
|-------|-------------|
| `NTraceAttachMode.OnFailure` | Attach traces to the job only on failure. |
| `NTraceAttachMode.Always` | Always attach traces to the job. |
| `NTraceAttachMode.Never` | Never attach traces to the job. |

## Usage

Reference values as `NTraceAttachMode.<Value>`, e.g. `NTraceAttachMode.OnFailure`.
