# NClickMode

`UiPath.UIAutomationNext.Enums.NClickMode`

How a click is performed before another action (e.g., before typing).

## Values

| Value | Description |
|-------|-------------|
| `NClickMode.None` | Do not click before the action. |
| `NClickMode.Single` | Single click before the action. |
| `NClickMode.Double` | Double click before the action. |

## Usage

Reference values as `NClickMode.<Value>`, e.g. `NClickMode.Single`.
