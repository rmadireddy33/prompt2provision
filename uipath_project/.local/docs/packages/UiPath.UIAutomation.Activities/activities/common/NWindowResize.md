# NWindowResize

`UiPath.UIAutomationNext.Enums.NWindowResize`

How the application or browser window is sized when it is initialized.

## Values

| Value | Description |
|-------|-------------|
| `NWindowResize.None` | No resize is applied to the application/browser. |
| `NWindowResize.Maximize` | The application/browser is maximized when initialized. |
| `NWindowResize.Restore` | The application/browser is resized and moved to the position and size it had when indicated. |
| `NWindowResize.Minimize` | The application/browser is minimized when initialized. |

## Usage

Reference values as `NWindowResize.<Value>`, e.g. `NWindowResize.Maximize`.
