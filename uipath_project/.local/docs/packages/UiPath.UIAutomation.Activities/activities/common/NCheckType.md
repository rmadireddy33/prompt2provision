# NCheckType

`UiPath.UIAutomationNext.Enums.NCheckType`

The action performed by the **Check/Uncheck** activity on a check box or radio button.

## Values

| Value | Description |
|-------|-------------|
| `NCheckType.Check` | Select a check box or radio button. |
| `NCheckType.Uncheck` | Clear a check box. |
| `NCheckType.Toggle` | Switch the state of a check box (select if cleared and clear if selected). |

## Usage

Reference values as `NCheckType.<Value>`, e.g. `NCheckType.Check`.
