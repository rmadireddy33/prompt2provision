# LimitType

`UiPath.UIAutomationNext.Models.ExtractData.LimitType`

The kind of limit applied to data extraction.

## Values

| Value | Description |
|-------|-------------|
| `LimitType.None` | No limit. |
| `LimitType.Rows` | Limit by number of rows. |
| `LimitType.Page` | Limit by number of pages. |

## Usage

Reference values as `LimitType.<Value>`, e.g. `LimitType.Rows`.
