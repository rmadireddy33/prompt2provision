# NClickType

`UiPath.UIAutomationNext.Enums.NClickType`

The type of click to perform.

## Values

| Value | Description |
|-------|-------------|
| `NClickType.Single` | Single click. |
| `NClickType.Double` | Double click. |
| `NClickType.Down` | Down click (mouse button press only). |
| `NClickType.Up` | Up click (mouse button release only). |

## Usage

Reference values as `NClickType.<Value>`, e.g. `NClickType.Single`.
