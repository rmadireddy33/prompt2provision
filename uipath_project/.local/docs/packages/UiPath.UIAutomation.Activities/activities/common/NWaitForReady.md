# NWaitForReady

`UiPath.UIAutomationNext.Enums.NWaitForReady`

Wait level for a target application to become ready before performing the action.

## Values

| Value | Description |
|-------|-------------|
| `NWaitForReady.None` | Does not wait for the target to be ready. |
| `NWaitForReady.Interactive` | Waits until only a part of the app is loaded. |
| `NWaitForReady.Complete` | Waits for the entire app to be loaded. |

## Usage

Reference values as `NWaitForReady.<Value>`, e.g. `NWaitForReady.Complete`.
