# BrowserUserDataFolderMode

`UiPath.UIAutomationNext.Enums.BrowserUserDataFolderMode`

How the browser user data folder is selected when launching a browser session.

## Values

| Value | Description |
|-------|-------------|
| `BrowserUserDataFolderMode.Automatic` | Automatic — let the activity choose. |
| `BrowserUserDataFolderMode.DefaultFolder` | Use the default browser profile folder. |
| `BrowserUserDataFolderMode.CustomFolder` | Use a custom folder specified via `UserDataFolderPath`. |

## Usage

Reference values as `BrowserUserDataFolderMode.<Value>`, e.g. `BrowserUserDataFolderMode.Automatic`.
