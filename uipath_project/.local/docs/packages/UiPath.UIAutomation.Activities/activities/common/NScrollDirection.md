# NScrollDirection

`UiPath.UIAutomationNext.Enums.NScrollDirection`

Scrolling direction for the **Mouse Scroll** activity.

## Values

| Value | Description |
|-------|-------------|
| `NScrollDirection.Up` | Scroll up. |
| `NScrollDirection.Down` | Scroll down. |
| `NScrollDirection.Left` | Scroll left. |
| `NScrollDirection.Right` | Scroll right. |

## Usage

Reference values as `NScrollDirection.<Value>`, e.g. `NScrollDirection.Down`.
