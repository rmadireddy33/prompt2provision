# NSemanticElementType

`UiPath.UIAutomationNext.Enums.NSemanticElementType`

The semantic element type used by AI-based target identification.

## Values

| Value | Description |
|-------|-------------|
| `NSemanticElementType.None` | Generic element. |
| `NSemanticElementType.Icon` | Icon. |
| `NSemanticElementType.Button` | Button. |
| `NSemanticElementType.CheckBox` | Checkbox. |
| `NSemanticElementType.DropDown` | Drop down. |
| `NSemanticElementType.InputBox` | Input. |
| `NSemanticElementType.InputBoxPassword` | Password input. |
| `NSemanticElementType.RadioButton` | Radio button. |
| `NSemanticElementType.Text` | Label. |
| `NSemanticElementType.DatePicker` | Date picker. |
| `NSemanticElementType.RadioGroup` | Radio group. |
| `NSemanticElementType.DateGroup` | Date group. |

## Usage

Reference values as `NSemanticElementType.<Value>`, e.g. `NSemanticElementType.Button`.
