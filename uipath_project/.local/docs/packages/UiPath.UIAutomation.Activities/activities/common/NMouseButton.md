# NMouseButton

`UiPath.UIAutomationNext.Enums.NMouseButton`

The mouse button to use for click operations.

## Values

| Value | Description |
|-------|-------------|
| `NMouseButton.Left` | Left mouse button. |
| `NMouseButton.Right` | Right mouse button. |
| `NMouseButton.Middle` | Middle mouse button. |

## Usage

Reference values as `NMouseButton.<Value>`, e.g. `NMouseButton.Left`.
