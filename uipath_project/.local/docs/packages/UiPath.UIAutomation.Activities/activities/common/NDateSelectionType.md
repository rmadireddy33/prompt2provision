# NDateSelectionType

`UiPath.UIAutomationNext.Enums.NDateSelectionType`

The selection mode used by SAP/calendar date pickers.

## Values

| Value | Description |
|-------|-------------|
| `NDateSelectionType.Date` | Select a specific day. |
| `NDateSelectionType.Range` | Select a period of time. |
| `NDateSelectionType.Week` | Select a week. |

## Usage

Reference values as `NDateSelectionType.<Value>`, e.g. `NDateSelectionType.Date`.
