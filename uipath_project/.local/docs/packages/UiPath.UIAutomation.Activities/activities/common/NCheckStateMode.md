# NCheckStateMode

`UiPath.UIAutomationNext.Enums.NCheckStateMode`

The state condition checked by the **Check App State** activity.

## Values

| Value | Description |
|-------|-------------|
| `NCheckStateMode.WaitAppear` | Wait for an element to appear. |
| `NCheckStateMode.WaitDisappear` | Wait for an element to disappear. |
| `NCheckStateMode.WindowNotActive` | Wait until the element's window is not active. |

## Usage

Reference values as `NCheckStateMode.<Value>`, e.g. `NCheckStateMode.WaitAppear`.
