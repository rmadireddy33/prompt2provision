# NBrowserDialogResponse

`UiPath.UIAutomationNext.Enums.NBrowserDialogResponse`

The response sent to a JavaScript browser dialog (alert, confirm, prompt).

## Values

| Value | Description |
|-------|-------------|
| `NBrowserDialogResponse.Cancel` | Cancel the dialog. |
| `NBrowserDialogResponse.OK` | Accept the dialog. |

## Usage

Reference values as `NBrowserDialogResponse.<Value>`, e.g. `NBrowserDialogResponse.OK`.
