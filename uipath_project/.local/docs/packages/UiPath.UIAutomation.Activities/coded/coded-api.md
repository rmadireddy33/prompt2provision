# UI Automation Coded API

- **Service accessor:** `uiAutomation`
- **Type:** `IUiAutomationAppService`
- **Package:** `UiPath.UIAutomation.Activities`

## Architecture

The UI Automation coded API is organized in two layers:

1. **Service layer (`IUiAutomationAppService`)** -- accessed via `uiAutomation`. Provides top-level operations such as `Attach`/`Open` (which return a `UiTargetApp` handle) and element-level actions that accept a [`UiElement`](../activities/common/UiElement.md) directly (e.g., `Click(UiElement)`, `TypeInto(UiElement)`). Also contains global utilities like `SetRuntimeBrowser`, `KeyboardShortcut`, `TakeScreenshot`, `GetClipboard`/`SetClipboard`, and `WaitState`.

2. **Handle layer (`UiTargetApp`)** -- returned by `Attach`/`Open`. Represents an attached or opened application window. Methods on this handle accept targets by `string` name (Object Repository target name), `IElementDescriptor`, `TargetAnchorableModel`, or `RuntimeTarget`. `UiTargetApp` implements `IDisposable` and should be disposed when no longer needed. It also exposes indexers to convert a target name or element descriptor into a `TargetAnchorableModel`.

Every action method has both a synchronous variant and an `Async` variant (suffixed with `Async`) that returns `Task` or `Task<T>`.

### External descriptors

`IElementDescriptor` and `IScreenDescriptor` are part of the Coded Workflows framework (`UiPath.CodedWorkflows.DescriptorIntegration` package), not of this API. Many methods accept them as parameters so that descriptors generated by the Coded Workflows tooling can be passed directly. Their schema is owned by that package; treat them here as opaque inputs.

### Runtime target models

`TargetAnchorableModel` is the runtime-model equivalent of [`TargetAnchorable`](../activities/common/Target.md#targetanchorable) — it carries the same selector, anchor, image, native text, semantic, and CV data that the activity-side `TargetAnchorable` exposes, but as a plain runtime object rather than an XAML sub-object with `InArgument<T>` properties. Most coded-API methods accept it via the `UiTargetApp` indexers (`uiTargetApp["target name"]` or `uiTargetApp[descriptor]`). See [`Target.md`](../activities/common/Target.md) for the full property surface.

`TargetAppModel` is the runtime-model equivalent of [`TargetApp`](../activities/common/Target.md#targetapp), used by `Attach(TargetAppModel)` / `Open(TargetAppModel)` for advanced Object Repository scenarios. `RuntimeTarget` is an opaque runtime handle (a `Guid` reference) returned by `GetRuntimeTarget` and consumed by other methods.

## Enum Types

Most enum types referenced by method signatures and option classes below are documented under [`../activities/common/`](../activities/common/). For each enum, the corresponding md file lists the available values and a description for each. Examples: [`NClickType`](../activities/common/NClickType.md), [`NMouseButton`](../activities/common/NMouseButton.md), [`NWaitForReady`](../activities/common/NWaitForReady.md).

Enums used only by this coded API (not by any activity) are documented inline at the end of this file: [`NScrapingOptions`](#nscrapingoptions).

The runtime element handle [`UiElement`](../activities/common/UiElement.md) and the verification options shape [`VerifyExecutionOptions`](../activities/common/VerifyExecutionOptions.md) are documented under `../activities/common/` since they are also referenced by activities.

## Common Patterns

### 1. Attach to a browser and click a button

```csharp
using var app = uiAutomation.Attach("MyWebApp");
app.Click("SubmitButton");
```

### 2. Open an application, type text, and read a result

```csharp
using var app = uiAutomation.Open("LoginScreen");
app.TypeInto("UsernameField", "admin@example.com");
app.TypeInto("PasswordField", new TypeIntoOptions
{
    SecureText = securePassword,
    EmptyFieldMode = NEmptyFieldMode.SingleLine
});
app.Click("LoginButton");

bool loaded = app.WaitState("Dashboard", NCheckStateMode.WaitAppear, 10);
string welcomeText = app.GetText("WelcomeLabel");
```

### 3. Extract table data from a web page with pagination

```csharp
using var app = uiAutomation.Open("ProductCatalog");
DataTable products = app.ExtractTableData("ProductsTable", new ExtractTableDataOptions
{
    LimitExtractionTo = LimitType.Rows,
    NumberOfItems = 100,
    DelayBetweenPages = 1.5
});

foreach (DataRow row in products.Rows)
{
    Log(row["ProductName"].ToString());
}
```

### 4. Use keyboard shortcuts and check element state

```csharp
using var app = uiAutomation.Attach("SpreadsheetApp");

app.Click("CellA1");
app.KeyboardShortcut("CellA1", "ctrl+c");
app.Click("CellB1");
app.KeyboardShortcut("CellB1", "ctrl+v");

bool isSaveEnabled = app.IsEnabled("SaveButton");
if (isSaveEnabled)
{
    app.Click("SaveButton");
}
```

### 5. Iterate child elements and get attributes

```csharp
using var app = uiAutomation.Attach("FileManager");

var children = app.GetChildren("FileList", new GetChildrenOptions
{
    SelectorFilter = "<webctrl tag='li' />",
    Recursive = false,
    Timeout = 10
});

foreach (var child in children)
{
    string fileName = app.GetText(child, new GetTextOptions
    {
        ScrapingMethod = NScrapingMethod.TextAttribute
    });
    string href = app.GetAttribute(child, new GetAttributeOptions { Attribute = "href" });
    Log($"File: {fileName}, Link: {href}");
}
```

## Enums

### NScrapingOptions

`UiPath.UIAutomationNext.Enums.NScrapingOptions`

Options for configuring text scraping behaviour. This is a `[Flags]` enum — values can be combined using bitwise OR.

| Value | Description |
|-------|-------------|
| `NScrapingOptions.None` | No options. |
| `NScrapingOptions.AllowFormatting` | Adds spaces and new lines to align the text as you see it on screen. Applies only to `NScrapingMethod.Native`. |
| `NScrapingOptions.IgnoreHiddenText` | Ignores text that is not visible. Visibility checks are performed using either accessibility or element bounds (ignoring empty area elements). Applies only to `NScrapingMethod.Fulltext`. |
| `NScrapingOptions.GetWordsCoordinates` | Enables word coordinates. Applies to `NScrapingMethod.Native` and `NScrapingMethod.Fulltext`. |
